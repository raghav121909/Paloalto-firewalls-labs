as part of my preparation for a Network Security Engineer role.

## Lab Environment
- Palo Alto VM-Series Firewall
- VMware Workstation

## Topics Covered
- Interface & Zone Configuration
- Security Policies
- NAT (Source & Destination)
- Traffic Monitoring
- Troubleshooting
