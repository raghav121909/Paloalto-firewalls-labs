# Palo Alto Firewall Labs

## Overview
This repository contains hands-on labs performed on a Palo Alto Next-Generation Firewall to build strong fundamentals in network security and firewall configuration.

All labs are executed in a VMware environment using the Palo Alto GUI and puTTY, following real-world deployment order and best practices.

---

## Lab Environment
- Palo Alto VM-Series Firewall
- VMware Workstation
- External Network: VMware NAT (Untrust)
- Internal Network: Ubuntu VM (Trust)
- Management: Web GUI and puTTY

---

## Labs Covered
- Untrust (ISP) Interface Configuration
- Trust Interface Configuration
- Static Routing
- Source NAT Configuration
- Security Policy Rules
- Website and Application Blocking
- Traffic Monitoring and Logs

---

## Key Skills Demonstrated
- Zone-based firewall architecture
- Layer 3 interface configuration
- Virtual routers and routing
- NAT and security policy implementation
- Traffic analysis and troubleshooting

---

## Purpose
This repository serves as a learning portfolio for Network Security Engineer roles and demonstrates practical experience with Palo Alto firewalls.
