# Lab 00 - Setup & Learning Mindset

## Objective
Build a real-world lab environment for Palo Alto Firewall learning.

## Why This Lab
- Understand what a Network Security Engineer does
- Learn using enterprise-grade firewall (Palo Alto)
- Practice using VM-based labs like real companies

## Lab Environment
- Palo Alto VM-Series Firewall (VMware)
- Ubuntu Linux (Internal Client)

## Outcome
- Palo Alto and Ubuntu successfully installed
- Lab ready for security configuration
